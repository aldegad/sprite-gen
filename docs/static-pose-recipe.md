# 정지 자세(Static-Pose) 행 레시피

수홍이 founder_v7 3방향 idle 을 직접 큐레이션한 데이터에서 도출한 규칙
(2026-07-19 확정 — "정지 idle 이잖아 사실 다? 누워있는 거 앉아있는 거,
물건 들고 서있는 것도. 안 뛰잖아. 액션도 없고.").

## 적용 대상

이동/액션이 아닌 모든 "가만히 있는" 상태: `idle`, `sit`, `sit_chair`, `lie`,
`carry_idle` 등. 로코모션(walk/run/jump)과 action 행에는 적용하지 않는다.

## 레시피 (실측 SSoT = founder_v7 down/side/up idle 큐레이션)

1. **정지 1컷 + 링크 복제**: 추출 프레임 중 최적 정지 포즈 1장을 고르고
   (사람 검수 대상; 기본 후보 frame 0), 나머지 시퀀스는 전부 그 프레임의
   **링크 복제**로 채운다 — 편집 truth 1곳, 아틀라스 셀 공유(무비용).
2. **길이 11컷 @ 4fps** (~2.75s 루프). 깜빡임이 붙는 정면은 +2컷까지 허용
   (down_idle 실측 13컷).
3. **호흡 레이어**: `breathe = {splits: [허리선], amplitude: 1, breaths: 3,
   subpixel: false}`. split 은 사슬로 정한다 — 같은 런에서 사람이 튜닝한
   형제 행 상속 → 벨트 위 허리선 검출 → 가슴 폴백 (`breathe.js`
   `waistSplitFrom`/`defaultBreatheConfig` 가 코드 SSoT). 실측: down/side
   0.62, up 0.59 (전부 벨트 바로 위).
4. **깜빡임은 눈이 보이는 방향만**: 시퀀스 **끝**에 배치.
   - 정면(down): 독립(unlink) 복제 1장에 감은-눈 픽셀 ops + 필요 시 원본
     깜빡임 프레임 1-2장 (실측 [28·독립 29ops, 6, 7]).
   - 옆(side): 독립 복제 1장에 눈 ops (실측 [13·독립]).
   - 뒤(up): 깜빡임 없음.
   감은-눈 픽셀은 **사람이 깎는다** — 자세마다 눈 위치가 달라 자동 이식
   금지 (포즈별 눈 좌표가 다르면 눈이 엉뚱한 데 감긴다).
5. **변형/픽셀 편집은 정지 원본 프레임에만** — 복제는 링크 동기화로 따라온다.

## 자동 적용 절차 (에이전트)

새 정지 자세 행이 추출되면 (큐레이션 없음 전제):

1. `curation.json` `states.<state>` 에: `selected=[still]`, `order=[still]+클론10`,
   `clones={새 idx: still}` (링크 기본 — 편집 복사 금지), breathe 위 레시피.
2. fps 4 를 `sprite-request.json` 해당 상태에 기록.
3. 깜빡임(정면/옆)은 자동 생성하지 않는다 — "깜빡임 추가 대기" 로 보고하고
   사람이 정지 프레임 확정 후 눈을 깎는다.
4. 열린 큐레이션 탭이 있으면 두-작성자 충돌 방지를 위해 새로고침을 안내한다.
